# Backend на JavaScript с базовыми функциями для магазина продуктов

## Описание
Серверная часть приложения, разработанная с использованием NodeJS и PostgreSQL. Содержит API для управления каталогом продуктов, создания и отслеживания статуса заказов. Пользователи могут регистрироваться, входить в систему, добавлять и удалять товары из корзины, оформлять заказы.

В зависимости от статуса пользователя прописаны контроллеры со следующими функциями:

* Для пользователя со статусом `user`:

  * просмотр каталога продуктов
  * добавление продукта в корзину
  * удаление продукта из корзины
  * просмотр корзины
  * оформление заказа
  * просмотр заказов
  * просмотр статуса выполнения заказа

* Для пользователя со статусом `admin`:

  * создание продукта
  * удаление продукта
  * просмотр заказов всех пользоватей
  * изменение статуса заказа всех пользователей

## Основные характеристики
*   **Создание базы данных:** Автоматическое создание базы данных postgres с заполнением категорий и брендов, подключение через пул соединений с использованием переменных окружения.
*   **Аутентификация и авторизация:**  Регистрация, вход в систему и защита маршрутов с использованием JWT (JSON Web Token) и ролей пользователей.
*   **Управление пользователями, каталогом продуктов и статусом заказов:**  Реализация различных асинхронных функций с помощью SQL-запросов с параметризацией и использование их после авторизации в зависимости от роли.
*   **Разграничение ролей:**  Авторизация на основе ролей пользователей (например, `user`, `admin`).

  
## Технологии

*   [NodeJS](https://nodejs.org/):  Среда выполнения JavaScript на стороне сервера, позволяющая создавать масштабируемые сетевые приложенияй.
*   [JWT (JSON Web Token)](https://jwt.io/):  Стандарт для создания токенов доступа.
*   [PostgreSQL](https://www.postgresql.org/): База данных, используемая для хранения данных.
*   [bcrypt](https://www.npmjs.com/package/bcrypt): Библиотека для хеширования паролей.
*   [dotenv](https://www.npmjs.com/package/dotenv):  Библиотека для загрузки переменных окружения из файла `.env`.
